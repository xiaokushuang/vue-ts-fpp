## 描述

- 拖拽组件

## 使用

```js
import PYDraggable from '@/components/PYDraggable/index.vue';

<PYDraggable> 
  <div>1</div>
  <div>2</div>
  <div>3</div>
</PYDraggable>
```

## 属性

| 参数 | 说明 | 类型 | 默认值 | 
| --- | --- | --- | --- |
| element | 外层包裹的元素 | String | div |
| options | draggable 列表配置项 | Object | null |
